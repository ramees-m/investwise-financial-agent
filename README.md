# 💹 InvestWise AI

**InvestWise AI** is an AI-powered financial analysis platform that evaluates companies and provides investment insights using a multi-agent system.

The application analyzes financial data, assesses risk levels, and generates investment recommendations through an interactive Streamlit dashboard.

---

## 🚀 Features

- 📊 Company financial data analysis
- 🤖 Multi-agent AI analysis system
- ⚠️ Risk level assessment
- 💡 Investment recommendations
- 📈 Financial metrics visualization
- 🧠 AI-generated summary insights
- 🖥 Interactive Streamlit dashboard

---

## 🏗 Architecture

The system uses a **multi-agent architecture** where different agents perform specialized financial tasks:

1. **Financial Data Agent**
   - Retrieves company financial information

2. **Analysis Agent**
   - Evaluates financial performance and metrics

3. **Risk Assessment Agent**
   - Determines investment risk level

4. **Investment Advisor Agent**
   - Generates investment advice

5. **Summary Agent**
   - Produces a final investment summary

---

## 🖥 Application Interface

The application contains two main pages:

### Input Page
- Enter company ticker
- Ask an investment question
- Run financial analysis

### Report Page
Displays:

- Financial Data
- Financial Analysis Metrics
- Risk Level
- Investment Advice
- AI-generated Summary
